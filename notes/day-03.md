# Day 03 — Strings, User Input, and Clean Output

## What I learned

1. String methods such as `strip()`, `lower()`, and `replace()` help clean messy user input.
2. Using `" ".join(text.split())` removes extra spaces between words.
3. A function lets me reuse the same ticket-title cleaning logic in more than one place.
4. Multiline f-strings make acknowledgements and formatted messages easier to read.
5. A `for` loop can process every request in a list without repeating code.

## Bug or issue fixed

I used `repr()` to check for hidden spaces in a string and confirmed that `strip()` removed the spaces at the beginning and end.
